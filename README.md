# AI PM 学习路径 · 全季酒店风单页

面向「新人转 AI PM」的系统化学习地图。全季酒店风格的温润视觉 + 渐进式学习路径 + 学前自测分流 + 成就证书，单文件纯前端，开箱即用。

## ✨ 功能亮点

- **渐进式学习路径**：12 周 / ~100h，4 个阶段 18 项任务，三态追踪（未开始 → 进行中 → 已完成）
- **6 维能力雷达自评**：产品感 / 技术理解 / 数据分析 / 项目管理 / AI 认知 / 沟通协作
- **职业自测 + 路径分支**：短测输出「技术型 / 业务型 / 双修型」倾向，高亮弱项并直接跳转到对应任务
- **动态进度仪表盘**：已完成 X/18、连续学习 X 天（含历史最佳）、本周 / 累计投入时长，随学习实时刷新
- **面试真题**：8 道 AI PM 高频面试题，全部带参考答案与加分项
- **完成证书**：阶段证书 + 总成就证书，可生成图片分享
- **就业前景数据**：基于智联招聘《2025 人工智能产业人才发展报告》+ 2026 趋势佐证
- **配套模块**：适合人群、术语词典、避坑指南、资源导航

## 🛠 技术栈

- 纯前端单文件 `index.html`（HTML + CSS + 原生 JavaScript，零依赖）
- 学习进度存浏览器 `localStorage`，无后端、无账号
- 字体：Noto Serif SC（标题）/ Noto Sans SC（正文）
- 视觉规范：温润米白 `#F5F1EA` + 木质暖棕 `#B89968` + 茶绿 `#8BA888`

## 🚀 本地预览

直接用浏览器打开 `index.html` 即可，无需构建。

## 📦 部署

通过 GitHub Pages 自动发布 `main` 分支根目录的 `index.html`。

## 🗂 数据键（localStorage）

| Key | 说明 |
|-----|------|
| `jj_tasks` | 18 项任务三态：0=未开始 / 1=进行中 / 2=已完成 |
| `jj_time_log` | `{ 日期: [{taskId, hours}] }` 每日完成记录与投入时长 |
| `jj_radar` | 6 维自评（0–10） |
| `jj_selfassess` | 职业自测答案与倾向分 `{answers, scores:{tech,biz}, ts}` |
| `jj_certificate` | 已生成证书记录 `{stage, genAt, shareId}` |
| `jj_notes` / `jj_badges` | 笔记 / 徽章 |

> 数据仅存本地浏览器，清空缓存会丢失；目前无跨设备同步。
